# Last Seconds - Godot Game

مشروع لعبة مطورة باستخدام محرك Godot 4.7

## معلومات المشروع
- **المحرك**: Godot 4.7
- **نوع اللعبة**: لعبة منصات (Platformer)
- **الحالة**: قيد التطوير

## المميزات الرئيسية
- شخصية لاعب قابلة للتحكم
- نظام تصادمات فيزيائي
- كاميرا تتابع اللاعب
- نقطة هدف للفوز
- إدارة الوقت والرسائل

## الملفات الرئيسية
- `project.godot` - إعدادات المشروع
- `World.tscn` - المشهد الرئيسي
- `player.gd` - سكريبت اللاعب
- `goal.gd` - سكريبت النقطة الهدف
- `game_manager.gd` - مدير اللعبة

## كيفية التشغيل
1. افتح Godot 4.7
2. افتح هذا المشروع
3. اضغط على زر التشغيل (Play)

## التحكم
- **الأسهم اليمين واليسار**: تحريك اللاعب
- **مسافة**: القفز
